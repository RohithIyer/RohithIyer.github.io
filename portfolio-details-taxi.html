Taxi Service Optimization

Back to Home

End-to-end system to forecast taxi demand and revenue, streamline fleet operations, and visualize KPIs.

Project Summary

  • Category: Operations / Data Science
  • Technologies: Python, XGBoost, RandomForest, Streamlit, Pandas
  • Data Source: NYC TLC Trip Record Data
  • Repo: GitHub

Overview

Built a modular forecasting and visualization system to help urban transport providers predict taxi demand, monitor revenue, and detect anomalies in fleet performance.

Problem Statement

Taxi operators face inconsistent demand, inefficient vehicle utilization, and limited visibility into operational performance.

Solution Approach

  • Trained machine learning models to predict demand (88.24% accuracy) and revenue (95.54% accuracy).
  • Built a real-time KPI dashboard using Streamlit with metrics for efficiency, utilization, and revenue per hour.
  • Implemented anomaly detection to flag underperforming routes or days automatically.
