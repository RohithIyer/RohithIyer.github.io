Airbnb Market Analysis (Austin)

Back to Home

Interactive Tableau dashboard analyzing Airbnb pricing, availability, and neighborhood trends in Austin, Texas.

Project Summary

  • Category: Market Research / Data Visualization
  • Technologies: Tableau, SQL, Python, Pandas
  • Dataset: Airbnb Austin (2016)
  • Repo: GitHub

Overview

Analyzed historical Airbnb data to uncover pricing insights, identify popular host zones, and visualize seasonal availability patterns for better decision-making by property owners and travelers.

Problem Statement

Short-term rental markets are highly dynamic, and hosts struggle to set competitive prices or optimize availability without data-driven insights.

Solution Approach

  • Processed and joined multiple Airbnb datasets (listings, calendar, reviews) using Python and SQL.
  • Built an interactive Tableau dashboard with filters for room type, price range, and location.
  • Performed temporal and spatial trend analysis to identify pricing outliers and demand hotspots.
