Overview

This project focused on building an end-to-end financial risk prediction platform that uses historical loan data to forecast both default probability and recommended interest rates.

Problem Statement

Lenders struggle to efficiently screen applicants at scale while balancing default risk and competitive interest rates. Traditional models are slow to adapt to new borrower behaviors or macroeconomic shifts.

Solution Approach

  • Engineered a dual-model pipeline combining XGBoost for binary classification (loan status) and LSTM for sequential risk prediction.
  • Processed and cleaned 50,000+ records using SQL and Python pandas; applied SMOTE for class balancing.
  • Deployed on Vertex AI for scalable model retraining and fast inference.
