💰 Financial Risk Prediction System

Built an advanced machine learning pipeline to predict loan default risk using structured customer and credit data.

🔧 Tools & Tech

  • Python (XGBoost, LSTM, pandas), SQL, MLflow for experiment tracking
  • Exploratory Data Analysis, feature engineering, model tuning

📊 Problem & Approach

This project aimed to improve loan approval decision-making by identifying high-risk applicants. Using a dataset of borrower financials, I trained multiple models and optimized hyperparameters using GridSearchCV. Both XGBoost and LSTM models were evaluated for ROC-AUC and precision-recall trade-offs.

📈 Key Outcomes

  • Achieved 92% classification accuracy using XGBoost
  • Reduced manual processing errors by integrating model predictions into an approval dashboard
  • Versioned and tracked experiments using MLflow
