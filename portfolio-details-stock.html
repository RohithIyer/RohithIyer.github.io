Ensemble-based system to forecast stock price trends using LSTM, sentiment analysis, and technical indicators.

Project Summary

  • Category: Finance / ML / Time Series
  • Technologies: Python, TensorFlow, yfinance, Dash, Redis
  • Accuracy: 87% (directional)
  • Repo: GitHub

Overview

This platform integrates deep learning, financial signals, and real-time data to predict directional stock movement and visualize market trends interactively.

Problem Statement

Investors often lack tools that combine technical analysis and sentiment insights in one place for short-term stock forecasting.

Solution Approach

  • Built LSTM, GRU, Transformer, and XGBoost models to capture price patterns.
  • Scraped and analyzed sentiment data from financial news and social platforms.
  • Integrated real-time data with yfinance and cached updates with Redis.
  • Used Dash to develop an interactive prediction dashboard.
